# walkerjacob.com — Personal Blog by Jacob Walker

[![GitHub stars](https://img.shields.io/github/stars/jacobwalker/walkerjacob.com?style=social)](https://github.com/jacobwalker/walkerjacob.com/stargazers)
[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/git/external?repository-url=https://github.com/jacobwalker/walkerjacob.com)

This is the codebase behind my personal blog [walkerjacob.com](https://walkerjacob.com) — a simple, fast, and clean site built with Next.js and Tailwind CSS. I use it to share thoughts, document projects, and post useful tools or ideas.

---

## ⚙️ Stack

- [Next.js](https://nextjs.org) — Framework
- [Tailwind CSS](https://tailwindcss.com) — Styling
- [Contentlayer](https://www.contentlayer.dev/) — Markdown + MDX content management
- [Vercel](https://vercel.com) — Deployment

---

## 🧠 Purpose

This blog is my creative and technical outlet. I built it from scratch to:

- Practice modern web dev (App Router, RSC, Tailwind 3)
- Document personal and dev-related projects
- Share resources or experiments I think others will find helpful

---

Clone the repo and get it running locally:

```bash

cd walkerjacob.com
